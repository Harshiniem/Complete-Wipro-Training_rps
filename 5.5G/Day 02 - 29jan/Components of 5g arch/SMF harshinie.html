N4 UPF Orchestrator

The primary link to the User Plane. Uses PFCP to create, modify, or release PDU sessions within the UPF.

N11 Connection to Access

Communication bridge to the AMF. Handles session management requests sent from the UE via NAS signaling.

N7/N10 Core Logic

N7 pulls policy rules from PCF, while N10 manages session-specific data from the UDM.

Key Use Case

When you start a video call, SMF allocates your IP address and sets the high-speed QoS parameters via the PCF (N7).
